# 🧙 AI Dungeon Master Adventure Game

An AI-powered text adventure game where you become the hero of a dynamic, story-driven world. The game uses **Google Gemini AI** to generate immersive storytelling, **Speech Recognition** for voice input, and **ElevenLabs** for text-to-speech narration.

---

## 🎮 Features

- 🧠 AI-generated dynamic storytelling (Gemini)
- 🎤 Voice-controlled gameplay (SpeechRecognition)
- 🧙 Real-time Dungeon Master narration (ElevenLabs TTS)
- ⚔️ Player stats system (health, inventory, story memory)
- 🗺️ Multiple adventure themes (Pirate, Fantasy, Space, Haunted)
- 🔁 Infinite branching story choices

---

## 🛠️ Tech Stack

- Python 3.10+
- Google Generative AI (Gemini)
- SpeechRecognition
- PyAudio
- ElevenLabs API
- Time module
